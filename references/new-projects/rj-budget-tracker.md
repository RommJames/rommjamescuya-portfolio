# RJ Budget Tracker

RJ Budget Tracker is a React web application for tracking personal income and expenses. It helps users monitor their balance, record transactions, and keep entries organized with filtering and sorting controls.

## Project Description

This project provides a simple and practical budgeting workflow:

- Add new income or expense transactions.
- Validate form inputs before saving transactions.
- View running totals for balance, total income, and total expenses.
- Edit existing transactions in a modal form.
- Delete transactions from the list.
- Filter transactions by type (`All`, `Income`, `Expense`).
- Sort transactions by date, amount, or description.

The app starts with sample transaction data to demonstrate the dashboard behavior immediately.

## Technologies Used

## Getting Started

### Prerequisites

- Node.js (LTS recommended)
- npm

### Installation

```bash
npm install
```

### Run Locally

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
